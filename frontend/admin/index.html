Admin

Operator tools and diagnostics for BalanceWhiz. Restrict access to this URL in production (for example via your host’s IP allowlist, VPN, or HTTP basic auth in front of the app).

API connectivity

Calls the public debug config endpoint (no secrets). Confirms the browser can reach the API configured for this deployment. 


              

            


            

              
Server-side operations

              

                Database maintenance and arbitrary SQL are only available through the API when MAINT_TOKEN is set in the server environment. Use POST /api/maintenance/sql with Authorization: Bearer <token> from a secure shell or automation—never from an untrusted browser extension.
              

              
Family-level admin (members, invites, data) is handled inside the app under Settings for users with the family admin role.
